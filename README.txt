Let your visitor share via their accounts on popular social networks such as Facebook, Google, Twitter, Pinterest And Linkedin

== Description ==

<h4>Sharing content</h4>

<p>The plugin currently supports the following networks for sharing your content:</p>

<ul>
<li>Facebook</li>
<li>Twitter</li>
<li>Google+</li>
<li>Pinterest</li>
<li>Linkedin</li>
</ul>

This plugin boasts some impressive features, it allows you to display share counts with template option.
<ul>
<li>Select Sharing Type.</li>
<li>Select Page To Show</li>
<li>Choose Place To Show</li>
</ul>

== Installation ==

1. Upload `pc-social-share.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go To Social Share setting page do changes and save.
4. You are done.
